Description

This class is a simple cookie manager. You create a cookie by creating the class.

Example: 

$cookie = new PWEL_COOKIE("MyCookieName");

Functions
  • addVar Adds a variable to the cookie
  • removeVar Removes a variable from the cookie
  • deleteCookieSession Deletes the cookie
  • getCookieVariables Returns an array of variables
addVar()
Description:
* Adds a variable to the cookie
* @param string $key
* @param string $value
Example: 
//Creates an instance with the cookie name
$cookie = new PWEL_COOKIE("MyCookieName");
//Adds a variable to the cookie
$cookie->addVar("var","Test");

var_dump($cookie->getCookieVariables());
Output: 
array
  'var' => string 'Test' (length=4)
removeVar()
Description:
* Removes the variable from the cookie
* @param string $key
Example: 
//Creates an instance with the cookie name
$cookie = new PWEL_COOKIE("MyCookieName");
//Removes a variable to the cookie
