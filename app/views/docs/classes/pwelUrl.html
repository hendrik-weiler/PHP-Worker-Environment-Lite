Description

This class is used for getting all parameters and doing redirections until now.

Functions
  • locateUrlVariables Returns all paramter from url
  • validateLink Returns a validated link for the framework
  • redirect Redirect to a given location
locateUrlVariables()
Description:
* Returns an array of all variables given in the url
* @return array

Example: 

//Create a PWEL_URL object
$url = new PWEL_URL();
//Store params into a variable
$params = $url->locateUrlVariables();
var_dump($params);

Output: 

array
  0 => string 'autoInject' (length=10)
  1 => string 'x' (length=1)
  2 => string '3' (length=1)
  3 => string 'a' (length=1)

validateLink()
Description:
* Returns a validated link
* @var string $file
* @return string
Example:
//Create a PWEL_URL object
$c = new PWEL_URL();
//Returns the validated path to a variable
$link = $c->validateLink("/class/method/param");
//print it out
print $link;
Output: 
/pwel/docs/class/method/param
redirect()
