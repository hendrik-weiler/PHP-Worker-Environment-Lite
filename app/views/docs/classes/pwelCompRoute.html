Description

Advanced component for routing.
Making custom routes possible.

Example: 

//Syntax = key:startvalue/key:startvalue
//For example this site your viewing at uses just
//the 'param' key with startvalue 'getStarted'
new PWEL_COMPONENT_ROUTE("lang:eng/class:/method:/param:");

//Displays all variables similiar to PWEL_URL's locateUrlVariables()
//but here the array got the key's you have given as parameter
//by creating the instance
var_dump(PWEL_COMPONENT_ROUTE::$variables)

// Contains a array of accepting redirections
// Example:
// lang default = eng
// The lang definitions are "de" and "eng"
// If someone try to look for "ita" he would be redirected to "eng"
// but with "de" and "eng" aswell
//
// With the range which can be set, the component accepting
// "de" or "eng"
// Scheme:
// array( $namespace = array($param,$param,$param,$param))
PWEL_COMPONENT_ROUTE::$acceptRange = array(
  "de","eng"
);
