Description

This class handles all selections from tables
similiar to a normal sql class which selects a specific area from tables

Functions
  • __construct Executing a search if parameter given
  • getColumns Returns an array of colum names of a table
  • getColumnNumbers Returns the number of column in a table
  • search Searches entries after pattern in a table
  • getRowIdBySearch Returns the entry position of a row as int value
  • getRowById Returns a row as array
__construct
Description:
Call a search and put it in $this->result
@param array $table
@param string $search

Example: 

$x = new eDB('../myNewDatabase');

//Creates an instance and search in 'tablename'
//table after everything '*'
$select = new eDB_Select('tablename', '*');

//Will contain a multidimensional array 
//using in foreach or for loops
var_dump($select->result);

@return array

search
Description:
Returns a search result as array

Search Patterns:
array($columName => $searchValue)

Advanced:
$columname => "<=1000"
$columname => ">=1000"
$columname => "<1000"
$columname => ">1000"
$columname => "!=1000"

@param string $tableName
@param array $search
@param string $column
@param string $sortWay
@return array

Example: 

$x = new eDB('../myNewDatabase');

$select = new eDB_Select();

//Contains a result of all entries where id is >= 1 
//and name isnt Peter
$result = $select->search('tableName',array(
   'id' => '>=1', 'name' => '!=Peter'
));
getRowIdBySearch
Description:
Returns the ID of the searched row
@param array $search
@param string $tablename
@return int

Example: 

$x = new eDB('../myNewDatabase');

$select = new eDB_Select();

//Contains the row entry position in table 'tablename'
$position = $select->getRowIdBySearch('tablename',array(
  'name' => 'Peter'
));
