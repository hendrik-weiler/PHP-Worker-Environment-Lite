Description

This class needs to be called everytime you wanna use an eDB class.

Functions
  • __construct Sets a new eDB working path
  • create Creates a new eDB database
  • delete Deletes a eDB database
__construct
Description:
Sets configuration
@param string $path

Example: 

      
        //Getting access to 'mydatabase'
        

        $x = new eDB('../mydatabase');
create
Description:
Creates a new db
@return bool

Example: 

      
        $x = new eDB('../myNewDatabase');
        

        

        //Creates a new database 'myNewDatabase'
        

        $x->create();
delete
Description:
Deletes the current db

Example: 

      
        $x = new eDB('../myNewDatabase');
        

        

        //Deletes the database 'myNewDatabase'
        

        //and all the tables within
        

        $x->delete();
